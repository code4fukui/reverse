# reverse

[
![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)
](https://opensource.org/licenses/ISC)

`msg.payload` の文字列表現を逆順にするシンプルな Node-RED ノードです。

## 機能

*   任意のペイロードを文字列に変換してから逆順にします。
*   文字列、数値、ブール値、その他のデータ型を処理できます。
*   元のメッセージ構造を維持し、`msg.payload` プロパティのみを変更します。
*   依存関係がゼロで、軽量にインストールできます。

## インストール

このノードをインストールするには、Node-RED のパレットマネージャーを使用するか、コマンドラインからインストールします。

### パレットマネージャーを使用する場合

1.  Node-RED エディタで「パレットの管理」メニューを開きます。
2.  「インストール」タブに移動します。
3.  `reverse` を検索します。
4.  このノードの「インストール」ボタンをクリックします。

### コマンドラインを使用する場合

1.  Node-RED のユーザーディレクトリ（例: `~/.node-red`）に移動します。
2.  以下のコマンドを実行します:
    ```bash
    npm install reverse
    ```
3.  Node-RED インスタンスを再起動します。

## 使い方

ノードパレットの `function` カテゴリから `reverse` ノードを見つけます。

このノードは任意のメッセージを受け取り、そのペイロードを逆順にします。入力された `msg.payload` を文字列に変換してから、文字単位で逆順にします。

### フローの例

1.  `Inject` ノード、`reverse` ノード、`Debug` ノードをワークスペースにドラッグします。
2.  `Inject` → `reverse` → `Debug` の順に接続します。
3.  `Inject` ノードで送信するペイロードを以下のように設定します:
    *   文字列: `"hello world"`
    *   数値: `12345`
4.  フローをデプロイし、`Inject` ノードのボタンをクリックします。
5.  デバッグサイドバーに逆順になったペイロードが表示されます:
    *   `"dlrow olleh"`
    *   `"54321"`

### 入力

*   `msg.payload` (`any`): 逆順にするデータ。ノードは自動的にこの値を文字列に変換します。

### 出力

*   `msg.payload` (`string`): 逆順になった文字列。
